# RealTimeBattle

This repository contains a copy of RealTimeBattle 1.0.8 patched to build on
modern systems.

For more details, please read the [original README] and visit the
[official website].

## Build

Install the following dependencies:

- build-essential
- pkg-config
- libgtk2.0-dev

Run:

```
autoreconf -i -f
./configure
make
```

## Run

Run:

```
./src/realtimebattle
```
